Track Changes (git)

  • Repository of all changes and versions
  • No fear of lost work, or undoing
  • Commit early and often; find your rhythm
  • Synch Local to Remote Repos (e.g. heroku, github)

Track Changes (git)

  • git init start tracking folder/project
  • git add files you want to track
  • git commit changes to keep them, w/message
  • git push changes from local -> remote repo
  • gittyup to add, commit, push -> heroku

Deploy (Heroku)

  • A service for running apps
  • Started with Rails, now runs anything
  • Create new, small apps for free
  • For beginners and experts
  • Start small, scale big
  • Has great docs, and online examples

Deploy (Heroku)

  • heroku create to start an app
  • git push to heroku starts deployment
  • gittyup shortcut to save and deploy
